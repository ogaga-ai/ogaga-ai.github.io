Enzyme Thermostability Prediction with ESM-2

PyTorch HuggingFace Transformers ESM-2 (150M) scikit-learn Ridge Regression Gradient Boosting UMAP NumPy pandas matplotlib

In enzyme engineering, most industrial enzymes lose efficiency above 40°C, but often, reactors are built to run at 60°C–80°C. This creates a temperature and usability problem. In this project, I built an ML model that predicts how stable an enzyme will be before performing any laboratory experiment.

My ML model uses Meta's ESM-2, a protein language model trained on 250 million sequences, to extract meaning from an amino acid sequence the way a language model extracts meaning from text.
